# publish
> 发布帖子相关页面的类

- [choose_circle 包](./choose_circle)

- PublishPostActivity
> 发布圈子帖子的页面Activity

- PublishPostComponent
> 发布圈子帖子的页面的dagger注入器

> 发布圈子帖子的一个接口类，便于管理presenter和view

- PublishPostFragment
> 发布圈子帖子的Fragment

- PublishPostPresenter
> 发布圈子帖子的presenter

- PublishPostPresenterModule
> 发布圈子帖子的module，提供view
