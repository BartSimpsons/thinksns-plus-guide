# messagelike
> 通知页面点击收到的赞进入的页面

- MessageLikeAdapter
> 收到的赞页面的adapter

- MessageLikeActivity
> 收到的赞页面的adapter

- MessageLikeComponent
> 收到的赞页面的dagger注入器

- MessageLikeContract
> 收到的赞页面的一个接口，便于管理Presenter和view

- MessageLikeFragment
> 收到的赞页面的Fragment

- MessageLikePresenter
> 收到的赞页面的Presenter

- MessageLikePresenterModule
> 收到的赞页面的module
