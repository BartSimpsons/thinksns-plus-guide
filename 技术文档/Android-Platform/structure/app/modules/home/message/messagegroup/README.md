# messagegroup
> 消息页面点击左上角，进入到的群聊列表的页面

- MessageGroupActivity
> 群聊列表的页面

- MessageGroupComponent
> 群聊列表的页面的dagger注入器

- MessageGroupContract
> 群聊列表的页面的一个接口，便于管理Presenter和view

- MessageGroupListFragment
> 群聊列表的页面的Fragment

- MessageGroupPresenter
> 群聊列表的页面Presenter

- MessageGroupPresenterModule
> 群聊列表的页面的module
