# gallery
> 画廊相关的类

- CustomImageSizeModel
> 获取Image的接口

- CustomImageSizeModelImp
> 获取Image的接口的实现类

- GalleryActivity
> 画廊Activity

- GalleryComponent
> 画廊界面的dagger注入器

- GalleryConstract
> 画廊界面的一个接口类，便于管理presenter和view

- GalleryFragment
> 画廊界面的Fragment（包含ViewPager的）

- GalleryPresenter
> 画廊界面的Presenter

- GalleryPresenterModule
> 画廊界面module

- GalleryPictureFragment
> 画廊界面ViewPager包含的Fragment，显示Image
