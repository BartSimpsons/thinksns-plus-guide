# init_password
> 初始化账号密码的页面，第三方登陆的账号，是没有密码的，需要先设置密码

- InitPasswordActivity
> 初始化密码页面的Activity

- InitPasswordComponent
> 初始化密码页面的dagger注入器

- InitPasswordContract
> 初始化密码页面的的接口类，便于管理presenter和view

- InitPasswordFragment
> 初始化密码页面的Fragment

- InitPasswordPresenter
> 初始化密码页面的presenter

- InitPasswordPresenterModule
> 初始化密码页面的module
