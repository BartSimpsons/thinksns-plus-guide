# music_play
> 音乐播放相关的类

- MusicPlayActivity
> 音乐播放界面的Activity

- MusicPlayComponent
> 音乐播放界面的dagger注入器

- MusicPlayContract
> 音乐播放界面的接口类，便于管理presenter和view

- MusicPlayFragment
> 音乐播放界面的Fragment

- MusicPlayPresenter
> 音乐播放界面的presenter

- MusicPlayPresenterModule
> 音乐播放界面的的module，提供view

- MusicPlayService
> 音乐的后台Service
