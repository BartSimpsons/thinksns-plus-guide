- QA_ListInfoFragment
> 问答的列表的Fragment

- QA_ListInfoFragmentPresenter
> 问答的列表的presenter

- QA_listInfoFragmentPresenterModule
> 问答的列表的module

- QAListBaseInfoAdapter
> 问答列表的base基类

- QAListInfoAdapter
> 问答列表的adapter

- SpanTextClickable
> 可点击textSpan，处理事件冲突

- TextRoundSpan
> 设置textmargin的span
